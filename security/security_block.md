    Note over 绑定邮箱业务,用户: 邮箱绑定拦截（在线绑定）
    绑定邮箱业务->>邮箱服务: 绑定邮箱前策略校验
    邮箱服务-->>绑定邮箱业务: 返回是否命中
    绑定邮箱业务-->>用户: 命中则拦截，绑定失败并提示兜底流程
